# markdown-comment

Markdown extension for human-AI collaborative commenting.

## Features

- 🎯 Inline annotations in Markdown
- 💬 Comment threads with replies
- 🤖 AI-friendly format
- ⚛️ React components ready

## Demo

👉 [Live Demo](https://huyansheng3.github.io/markdown-comment/)

## Packages

| Package | Description |
|---------|-------------|
| `@comment-md/core` | Core parser and serializer |
| `@comment-md/remark-plugin` | Remark plugin for annotation syntax |
| `@comment-md/react-ui` | React UI components |

## Quick Start

```bash
pnpm install
pnpm dev
```
