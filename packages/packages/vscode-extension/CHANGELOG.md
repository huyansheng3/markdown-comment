# Changelog

All notable changes to the "Markdown Comments" extension will be documented in this file.

## [0.1.0] - 2026-03-04

### Added

- Initial release
- Markdown outline navigation with hierarchical tree view
- Comment system based on comment-md syntax
- Add comments to selected text
- View comments in sidebar panel
- Highlight annotated content in editor
- Resolve/archive comment threads
- Configurable highlight colors
